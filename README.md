**LeNet** 实现

数据集使用 **MNIST**`

| 文件         | 描述                   |
| ----------- | -------------------- |
| `model.py`  | 定义LeNet模型架构。         |
| `train.py`  | 训练和验证的脚本，拆分数据、模型评估~。 |
| `test.py`   | 加载最佳模型并评估其在测试集上的表现。  |
| `utils.py`  | 数据加载和预处理的辅助函数。       |
| `config.py` | 超参数的配置。              |
